DeepMars

Train a convolutional neural network in your browser to classify imagery from Mars into three different classes. Use the Model Designer to build a custom architecture and then train it using the Model Trainer. The dataset will be downloaded in the background for you. Refresh the page to completely reset.

Dataset

The dataset you'll be using is comprised of 50x50 pixel grayscale images that were downlinked from NASA's Mars Exploration Rover (MER) in 2005. Someone has already labelled these images as "sand", "rocks", or "bedrock". The dataset has been split into a train and test set.
